QueryNest — Knowledge Discovery & Internal Search System

This workspace contains a MERN stack project scaffold for the "Knowledge Discovery & Internal Search System".

Folders:
- `frontend/` — Vite + React (Tailwind) frontend
- `backend/` — Express + Node backend

See `./.env.example` for environment variables.

Dev notes
- Run frontend: `cd frontend && npm install && npm run dev`
- Run backend: `cd backend && npm install && npm run dev`

Follow the detailed README sections in each folder for further steps.
