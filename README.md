# QueryNest



A full-stack document management system with OCR-powered text extraction and intelligent search.

## Tech Stack

- __Frontend__: React (Vite), Tailwind CSS
- __Backend__: Node.js/Express, MongoDB/GridFS
- __OCR__: Tesseract.js for scanned PDFs
- __Deployment__: Render (backend web service, frontend static site)

## Features

- User auth (JWT), document upload (PDF/DOCX/TXT/PPTX)
- Text extraction (parsing + OCR for scanned docs)
- Full-text search (titles, categories, tags, content)
- Dark theme UI, responsive design
- Real-time document previews with search snippets

## Usage

- Signup/login, upload docs with metadata
- Search across files with highlighted snippets
- View extracts, download, manage versions
